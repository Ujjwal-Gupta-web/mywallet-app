# MyWallet

### FEATURES
<ul>
<li>ADD/WITHDRAW money in wallet</li>
<li>SEND money to contacts</li>
<li>GET cashbacks</li>
<li>EMAIL based OTP and transaction alerts</li>
<li>Transaction History</li>
</ul>

### TECH STACK

|  |  |
|----------|----------|
| Frontend   | React, Redux   |
| Backend   | Java, Spring Boot   |
| Database   | MongoDB   |
| Unit Testing   | Jest, JUnit   |
